# Portfolio — Felipe Souza Moreira

Portfolio pessoal desenvolvido com HTML, CSS e JavaScript puro, apresentando habilidades, certificações, projetos e experiências profissionais.

**Acesse:** [felpes513.github.io/Portfolio](https://felpes513.github.io/Portfolio/)

---

## Seções

- **Início** — Apresentação e links sociais (GitHub, LinkedIn, Email)
- **Skills** — Tecnologias agrupadas por categoria
- **Certificações** — Badges verificáveis via Credly
- **Projetos** — Projetos com links para repositório, demo e vídeo
- **Experiências** — Histórico profissional em linha do tempo

## Tecnologias utilizadas

| Categoria | Tecnologias |
|---|---|
| Backend | Java, Spring Boot |
| Frontend | Angular, HTML5, CSS3, JavaScript, TypeScript |
| DevOps / Cloud | Docker, AWS, GitHub Actions |
| Banco de Dados | MySQL, MongoDB |
| Testes / QA | Selenium, Postman, Insomnia |

**Stack do projeto:**
- HTML5, CSS3 (custom properties, flexbox, grid)
- JavaScript ES6+ (vanilla, sem frameworks)
- Google Fonts (Roboto) e Material Icons via CDN

## Funcionalidades

- Layout responsivo com sidebar retrátil (hamburger menu no mobile)
- ScrollSpy via `IntersectionObserver` para destacar seção ativa na navegação
- Smooth scroll entre seções
- Acessibilidade com atributos ARIA
